# Duração
Ficou acordado que o projeto será desenvolvido contando com o tempo que cada desenvolvedor tera para desenvolver, levando em consideração que essa tarefa é um diferencial, sendo que ,preferencialmente o foco de nosso trabalho se desenvolve em atividades do escopo de cada laboratorista.

# Objetivo
Criar uma features que tenha todas as informações de todos os equipamentos mantidos pelo nucleo de suporte técnico mackenzie e nos laborátorios de informatica.
* Ativos laboratorios
* Ativos salas de aula
* Ativos Administrativos (salas de coordenação e professores).

<strong>Dado que:</strong>
	<p>Hoje os técnicos utilizam planilhas de excel para documentar os dados de ativos.</p>

<strong>Quando:</strong>
	<p>Precisam organizar as informações para gerar relatorios pedidos de ultima hora ou para manter documentado os equipamento sendo que muitas vezes computadores se perdem e acaba saindo do escopo geral de ativos.</p>

<strong>Então:</strong>
	<p>Elaboramos uma aba no sistema para melhor gerir esses dados, onde todos poderão vizualizar os equipamentos e adicionar em seus respectivos dominios(definidos no escopo de serviço). </p>

# Mapa de Ações da features
* CRUD completo das informações de cada equipamento.
* Geração de relatórios cvs/pdf.
